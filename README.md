# ConversorMoedas
Um simples conversor de moedas consumindo awesomeapi gratuita !

# Primera tela sem Interação
> Digite o valor que deseja converter supeior a "0".
> 
  > Escolha a "Moeda" que será convertida.
> 
   >   Clique em "Calcular".
